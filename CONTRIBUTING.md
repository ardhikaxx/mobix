# Contributing to Mobix

Terima kasih tertarik berkontribusi ke Mobix! 🎉

## Cara Berkontribusi

### Melaporkan Bug
Buka [issue](https://github.com/ardhikaxx/mobix/issues) baru dengan:
- Judul yang jelas
- Langkah reproduksi
- Screenshot (jika ada)
- Lingkungan (browser, OS)

### Mengajukan Fitur
Buka [issue](https://github.com/ardhikaxx/mobix/issues) dengan label `enhancement` dan jelaskan:
- Fitur yang diinginkan
- Kenapa fitur ini berguna
- Contoh implementasi (opsional)

### Pull Request
1. Fork repo ini
2. Buat branch: `git checkout -b feat/namafitur`
3. Commit perubahan
4. Push: `git push origin feat/namafitur`
5. Buka Pull Request

### Development Setup
```bash
git clone https://github.com/ardhikaxx/mobix.git
cd mobix
npm install
npm run dev
```

### Stack
- **Next.js** (App Router) — framework
- **TypeScript** — bahasa
- **Tailwind CSS** — styling
- **Firebase** — auth, database, hosting
- **SWR** — data fetching

## Kode Etik
Dengan berkontribusi, kamu setuju untuk mengikuti [Code of Conduct](CODE_OF_CONDUCT.md) kami.

## Lisensi
Dengan mengirimkan kontribusi, kamu menyetujui bahwa kontribusimu akan dilisensikan di bawah lisensi yang sama dengan proyek ini.
