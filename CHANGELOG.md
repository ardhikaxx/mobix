# Changelog

## [1.0.0] - 2026-07-10

### Added
- Home page with app grid, search, categories, and Load More
- App detail page with screenshots, description, and download
- User authentication (email + Google)
- Review and rating system via Firestore
- Share dialog with social media + embed codes (HTML/Markdown/BBCode)
- Developer page with WhatsApp + QRIS payment flow
- Related apps section on detail page
- Pagination (Load More) on home page
- App filtering by category and sorting (newest/popular)
- Download counter per app via Firestore
- Search with debounce
- Loading skeletons for better UX
- PWA support (manifest + service worker)
- Dynamic OG Image for social sharing
- Sitemap and robots.txt for SEO
- Breadcrumb JSON-LD + hreflang tags
- Dark mode support
- i18n Indonesian / English
- Mobix Badge (SVG + React component)
- TrustBadge with avatar group + user count from Firestore
